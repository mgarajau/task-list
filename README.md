# 📋 Lista de Tarefas

Uma aplicação de lista de tarefas desenvolvida com foco em praticar conceitos fundamentais do desenvolvimento Front-End.  
O projeto permite adicionar, concluir e remover tarefas de forma simples, rápida e intuitiva.

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## 💡 Conceitos aplicados

Durante o desenvolvimento deste projeto, foram aplicados diversos conceitos importantes:

- **Manipulação do DOM** para criar interações dinâmicas
- **Manipulação de elementos HTML via JavaScript**
- **Alteração de estilos CSS via JavaScript**
- **Local Storage** para salvar as tarefas no navegador, mantendo os dados mesmo após atualizar a página
- **Eventos JavaScript** para capturar interações do usuário
- **Responsividade** utilizando CSS para adaptação em diferentes tamanhos de tela

## ⚙️ Funcionalidades

✔ Adicionar novas tarefas  
✔ Marcar tarefas como concluídas  
✔ Remover tarefas  
✔ Persistência de dados com Local Storage  
✔ Interface adaptável para desktop, tablet e smartphone  

## 🎯 Objetivo do projeto

Este projeto foi desenvolvido com o objetivo de fortalecer conhecimentos em desenvolvimento Front-End, principalmente em:

- Estruturação com HTML
- Estilização com CSS
- Lógica de programação com JavaScript
- Manipulação dinâmica da interface

## 📱 Responsividade

A aplicação foi desenvolvida seguindo o conceito de **Mobile First**, garantindo uma boa experiência em diferentes dispositivos.
